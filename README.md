شرح پروژه

Advanced List Tool یک ابزار خط فرمانی (CLI) و ماژولار برای مدیریت، تحلیل و انجام عملیات پیچیده روی دو لیست پایتون است. این پروژه برای اهداف آموزشی، توسعه و استفاده در اسکریپت‌های تحلیلی طراحی شده و تمرکز آن بر پایداری، قابلیت توسعه و مدیریت امن ورودی‌هاست.

اهداف کلیدی

پذیرش ورودی‌های متنوع (اعداد صحیح/اعشاری/منفی، بولین، None، رشته‌های quoted، literal پایتون)

عملیات متناظر (element-wise) با حالات مختلف (zip / zip_longest) و مدیریت ناسازگاری نوع‌ها

مجموعه‌ای کامل از عملیات (جمع/تفریق/ضرب/تقسیم متناظر، append/extend/concatenate، set ops، sorting، unique، reverse)

ذخیره/بارگذاری امن JSON، تاریخچهٔ مختصر (undo)، و طراحی ماژولار برای توسعهٔ بعدی



ویژگی‌ها (Features)

ورودی امن و قابل‌اعتماد با ast.literal_eval و پردازش تدریجی

عملیات قابل انتخاب از منوی تعاملی

پشتیبانی از حالت‌های strict و fallback در عملیات متناظر

گزینه‌های مرتب‌سازی: رشته‌ای، عددی، گروه‌بندی بر اساس نوع

تابعیت درونی برای تبدیل/تحلیل انواع (helper utilities)

ذخیره/بارگذاری در قالب JSON

Undo یک‌مرحله‌ای (snapshot)

طراحی شیءگرا و ماژولار (ListManager) برای استفادهٔ برنامه‌ریزی‌پذیر
